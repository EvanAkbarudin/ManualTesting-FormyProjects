# Formy Manual Testing

## 📌 Project Overview

This repository contains the manual testing documentation for the **Formy Demo** web application. The project focuses on validating the application's core functionalities through structured manual testing to ensure that each feature works as expected.

The testing process includes creating test scenarios, documenting test cases, executing test cases, and recording test execution results.

---

## 🎯 Testing Scope

The following features were tested:

* Complete Web Form
* Input Fields
* Dropdown Menu
* Radio Button
* Checkbox
* Date Picker
* File Upload
* Drag and Drop
* Keyboard & Mouse Interaction
* Form Submission
